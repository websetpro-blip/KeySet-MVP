# План исследования: Библиотеки экспорта/импорта файлов в браузере

## Цель
Исследовать актуальные библиотеки для экспорта/импорта файлов в браузере (CSV, XLSX, TXT) для веб-версии KeySet проекта.

## Критерии оценки
- Простота использования
- Размер bundle
- Поддержка форматирования (стили ячеек в Excel)
- Актуальность (2025)
- Поддержка кириллицы (UTF-8)
